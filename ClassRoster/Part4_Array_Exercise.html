Welcome! Please answer the following prompts to create a class roster:

This is a very simple web app that contains an array of student names in a .js file. The website will prompt you for four possible commands to alter your student roster: add,remove,display, and quit. This web page will prompt you for one of the commands. Here is what each command should do:

  • add: will prompt you for a student name. The name will then be added to the student to the array of student names in the .js file
  • remove: will cprompt you for the student name you wish to remove, then remove this name from the roster array
  • display: will print out the roster
  • quit:will end the editing period for your class roster.

This is a simple app where the names will appear in the console (to view simply right click, click 'inspect' in the drop down, and click the tab for 'Console') and will not be readily accessible elsewhere.

It is important to note if you are reviewing these instructions above as you can NOT see the displayed class roster in your console (please see above for instructions on how to open the console), refresh the page with the Developer Tools Console (as described above) already open rather than opening the console once the app is already running.
